# ✈️ Skyfire Uprising – Level 4

## 🎮 Overview
This project simulates a **realistic aerial combat experience**.  
Players take control of a fighter jet equipped with **machine guns and homing missiles**, navigating through diverse environments such as **oceans, cities, and canyon deserts**.

Your ultimate objective is to **guide a heavy cannon strike** to destroy the alien command ship dominating the skies, unlocking the next chapter of the game.
